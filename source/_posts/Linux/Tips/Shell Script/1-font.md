## 特殊字元

| 符號         | 名稱             | 意義                         |
| :----------- | :--------------- | :--------------------------- |
| $            | 變數             | 取變數名稱對應的值           |
| #            | 註解             |                              |
|              | 跳脫符號         | 特殊及萬用字元還原成一般字元 |
| \|           | 管線             |                              |
| ;            | 分隔連續指令符號 |                              |
| $            | 工作控制         | 將指令變成背景執行           |
| /            | 目錄符號         | 分隔路徑                     |
| >,>>         | 資料重導向(輸出) |                              |
| <，<<        | 資料重導向(輸入) |                              |
| ‘’           | 單引號           | 無法取變數值                 |
| “”           | 雙引號           | 可以取變數值                 |
| `\` 或是 $() | 裏面包執行的指令 |                              |
| ()           |                  | 子 shell的始末               |
| {}           | 中間爲命令區塊   |                              |

## 萬用字元

| 符號 | 功能                                                      |
| :--- | :-------------------------------------------------------- |
| *    | 0到無窮多個任意字元                                       |
| ?    | 一個以上的任意字元                                        |
| []   | 一個以上括號內的字元                                      |
| -    | 編碼範圍內的所有字元，例如[0-9]代表0到9之間的所有數字     |
| ^    | 後面接的字元都不要，例如[^0-9]代表0-9之間的所有字元都不接 |
